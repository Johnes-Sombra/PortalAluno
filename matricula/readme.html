Introdução

Esta é a documentação para o projeto "Ficha de Matrícula do Estudante". O projeto consiste em um formulário HTML para coletar informações de matrícula de estudantes, com validações em JavaScript e possibilidade de processamento dos dados em um servidor PHP para armazenamento em um banco de dados MySQL.

Arquivos do Projeto

  • index.html: Arquivo principal contendo o formulário de matrícula.
  • estilo-formulario.css: Arquivo CSS para estilizar o formulário e a página.
  • ficha-matricula.js: Arquivo JavaScript para adicionar interatividade ao formulário.
  • processar_formulario.php: Script PHP para processar os dados do formulário e inseri-los no banco de dados MySQL.
  • readme.html: Este arquivo, contendo a documentação do projeto.

Configuração e Funcionamento

Para configurar e fazer o projeto funcionar corretamente, siga estas etapas:

  1. Faça o download dos arquivos do projeto para o seu ambiente de desenvolvimento.
  2. Configure um servidor web local com suporte a PHP e MySQL. Você pode usar ferramentas como XAMPP, WAMP ou MAMP para isso.
  3. Crie um banco de dados MySQL e uma tabela para armazenar os dados do formulário.
  4. Atualize as credenciais do banco de dados no arquivo processar_formulario.php.
  5. Abra o arquivo index.html em um navegador web para acessar o formulário de matrícula e preencha os campos necessários.
  6. Envie o formulário para processamento e verifique se os dados foram corretamente inseridos no banco de dados.

Com essas etapas concluídas, o projeto estará configurado e funcionando corretamente, permitindo a coleta e armazenamento dos dados de matrícula dos estudantes.

Para qualquer dúvida ou problema, consulte a documentação ou entre em contato com o administrador do sistema.
